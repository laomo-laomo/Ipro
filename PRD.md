## 1. 产品概述

### 1.1 产品名称
**Ipro** — 智能童话绘本生成平台

### 1.2 一句话描述
用户上传自己的照片，AI 将其变成童话故事主角，自动生成图文插画册和 MP4 视频。

### 1.3 核心价值
- 把「普通照片」变成「童话主角」
- 个性化故事体验（自己的脸出现在故事里）
- 全自动生成，无需设计技能

### 1.4 目标用户
- 3-12 岁儿童家长（送给孩子的个性化礼物）
- 早教机构（批量生成教学内容）
- 内容创作者（小红书/抖音素材）

## 2. 功能模块

### 2.1 用户系统

| 功能 | 说明 |
|------|------|
| 注册登录 | 微信一键登录、手机号登录 |
| 微信登录 | 公众号网页授权 |
| 会员管理 | 查看会员状态、到期时间、剩余额度 |

### 2.2 角色生成

| 功能 | 说明 |
|------|------|
| 照片上传 | 支持 JPG/PNG，建议 512x512 以上 |
| 角色特征提取 | 自动识别面部特征、发型、肤色等 |
| 角色风格化 | ChatGPT Images 2.0 Edit，风格化成插画主角 |
| 风格预览 | 用户确认预览，可重新生成调整 |
| 角色存储 | 保存风格化后的主角形象（有效期与账号绑定） |

### 2.3 故事创建

| 功能 | 说明 |
|------|------|
| 选择预设模板 | 小红帽、白雪公主、三只小猪等经典童话 |
| 输入童话名称 | 用户输入名称，LLM 生成完整故事 |
| 故事预览 | 展示故事全文和每段场景描述 |
| 故事编辑 | 用户可调整故事文字（可选） |

**预设童话列表（第一批）：**
1. 小红帽
2. 白雪公主
3. 三只小猪
4. 灰姑娘
5. 睡美人
6. 丑小鸭
7. 匹诺曹
8. 美女与野兽

### 2.4 图文插画生成

| 功能 | 说明 |
|------|------|
| 场景拆分 | 自动将故事拆分为 N 个场景 |
| 批量插画生成 | For 每个场景：①生成场景背景 ②角色+背景合成 |
| 角色一致性 | 所有场景复用同一风格化主角形象 |
| 换装支持 | 不同场景主角服装变化 |
| 生成进度 | 显示当前生成到第几页，支持失败重试 |

**插画生成流程：**
```
Step 1: 角色风格化（每套故事只需一次）
    用户上传照片
        ↓
    ChatGPT Images 2.0 Edit → 风格化成插画角色
        ↓
    得到「插画版主角形象」

Step 2: 批量生成故事场景
    For 每个场景：
        ① ChatGPT Images 2.0 → 生成场景背景
        ② ChatGPT Images 2.0 Edit（①垫图 + 插画角色合成）
        → 得到「主角在场景中」的完整插画
```

**计费规则：0.2 元/张，后台可调整**

### 2.5 图文插画册

| 功能 | 说明 |
|------|------|
| 自动排版 | 插画 + 故事文字，自动换行 |
| 预览 | 在线预览完整绘本 |
| 下载 | 导出 PDF / 高清图片合集 |
| 付费下载 | 非会员按张扣费 |

### 2.6 MP4 视频生成

| 功能 | 说明 |
|------|------|
| 配音选择 | 默认音色（Edge TTS）/ 克隆音色 |
| 字幕生成 | 自动识别文字断句，生成字幕 |
| 字幕高亮 | 关键字 1.5x 大小 + 亮色 |
| 动态效果 | Ken Burns / 缩放 / 平移 |
| Remotion 渲染 | 音频画面对齐，导出 MP4 |
| 输出规格 | 1920x1080，30fps，MP4 |

**计费规则：**
- 默认音频（Edge TTS）：免费
- 使用克隆音色：0.2 元/千字，后台可调整

### 2.7 声音克隆

| 功能 | 说明 |
|------|------|

## 5. 技术架构

### 5.1 技术栈

| 层级 | 选型 | 说明 |
|------|------|------|
| 前端 | Next.js 15 + TypeScript | SSR + 静态导出兼顾 |
| UI | TailwindCSS + shadcn/ui | 快速出活，组件丰富 |
| 后端 | Node.js + Fastify | 高性能 API |
| 数据库 | SQLite（开发）/ PostgreSQL（生产） | 数据持久化 |
| AI 图像 | apiz.ai（ChatGPT Images 2.0 / 2.0 Edit） | 更便宜，支持 Webhook 回调 |
| AI 语音 | Edge TTS + MiniMax | 配音 + 声音克隆 |
| 视频渲染 | Remotion | MP4 导出 |
| 存储 | 本地 + 阿里云 OSS | 文件存储 |
| 部署 | Docker + 阿里云 | 可迁移 |

## 8. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| ChatGPT Images API 涨价/不可用 | 成本上升/服务中断 | 预留其他图像 API 作为备选（Midjourney、Stable Diffusion） |
| 声音克隆版权问题 | 合规风险 | 明确用户协议，声音仅限本人使用 |
| 内容审核 | 涉黄涉暴内容 | 增加 AI 内容过滤层 |
| 并发量高 | 生成队列积压 | 消息队列削峰，限流 |
| 微信支付商户资质 | 无法收款 | 提前申请商户号 |
