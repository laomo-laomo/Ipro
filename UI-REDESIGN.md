> 版本：v1.0

## 1. 现状分析

### 1.1 当前 UI 技术栈
- Next.js 15 (App Router) + TypeScript
- TailwindCSS 3.x + shadcn/ui (Radix)
- lucide-react 图标库
- 响应式布局（移动端底部导航 + 桌面端顶部导航）

### 1.2 现有问题

| 问题 | 严重程度 | 说明 |
|------|---------|------|
| **无品牌视觉识别** | 高 | 使用默认 shadcn 蓝色主题，与童话/绘本产品气质完全不搭 |
| **首页缺乏冲击力** | 高 | 仅 4 张 emoji 卡片，无 Hero 区、无视觉叙事、无行动引导 |
| **模板卡片用纯色块** | 高 | 8 个童话模板全部是 g-red-200 等纯色背景，无插画封面 |
| **缺少动画与微交互** | 中 | 无页面过渡、无加载骨架屏动画、无 hover 微动效 |
| **创作流程页面重复** | 中 | 4 个步骤页面结构雷同（header + stepper + 内容区），缺乏节奏感 |
| **色彩体系冷淡** | 高 | 冷蓝色调（--primary: 221.2 83.2% 53.3%），应为温暖梦幻色系 |
| **导航缺乏个性** | 中 | 顶部/底部导航功能完整但无品牌感，logo 仅为文字 |
| **会员页缺乏说服力** | 中 | 标准定价表，无价值锚点、无社交证明、无视觉层级 |
| **空状态设计单调** | 低 | 统一灰色图标 + 文字，缺乏情感化设计 |
| **插画卡片缺乏层次** | 中 | 作品卡片布局平淡，封面图占比不够突出 |

## 2. 设计目标

### 2.1 核心设计理念
**「打开一本属于你的童话书」**

用户每一次操作都像翻阅一本精装童话绘本 — 从封面到扉页，从故事到插画，最终合上一本属于自己的书。

### 2.2 设计原则

| 原则 | 说明 |
|------|------|
| **魔法感** | 每一次交互都应有「魔法发生」的感觉，AI 生成过程是核心体验时刻 |
| **绘本质感** | 整体视觉风格模拟精装绘本 — 圆角、纸张纹理、温暖色调、手绘元素 |
| **引导式叙事** | 用户不是在「操作功能」，而是在「讲述一个故事」，流程应有叙事节奏 |
| **移动优先** | 核心用户（家长）以手机使用为主，所有设计以移动端体验为第一优先 |
| **情感化反馈** | 加载、成功、错误状态都应有情感表达，而非冰冷的技术提示 |

**现状问题：** 功能完整但无品牌感，logo 仅为文字 + 图标

**重设计方案：**

│  ✨ IPro            [首页] [创作] [作品] [👤] │  ← 桌面端
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  [🏠首页]  [✨创作]  [📖作品]  [👤我的]     │  ← 移动端底部
└─────────────────────────────────────────────┘
`

**改动要点：**
- Logo 区域增加品牌图标（魔法书/星星）+ 渐变色文字
- 移动端底部导航从 5 项精简为 4 项（合并「声音」和「会员」到「我的」）
- 激活状态使用品牌紫渐变底部指示条，而非简单变色
- 未登录时显示「登录」按钮带微光动效
- 导航栏背景增加轻微毛玻璃效果 + 暖色底纹

### 5.3 创作流程重设计

**现状问题：** 4 个步骤页面结构完全相同（header + stepper + 内容），缺乏节奏感

**重设计方案 — 「翻书」隐喻：**

#### 全局改动
- 步骤指示器从「横排胶囊标签」改为「绘本页码」样式：① ② ③ ④
- 每个步骤页面使用不同的暖色渐变背景（淡紫 → 淡粉 → 淡金 → 淡绿）
- 页面切换增加「翻页」过渡动画（CSS transition）
- 移除每个页面重复的 <h1> 标题，统一由步骤指示器承载

## 6. 动效规范

### 6.1 页面过渡

`css
/* 页面切换 — 翻书效果 */
.page-enter {
  opacity: 0;
  transform: translateX(30px);
}
.page-enter-active {
  opacity: 1;
  transform: translateX(0);
  transition: all 0.3s ease-out;
}
`

### 6.2 微交互动效

| 场景 | 动效 | 时长 |
|------|------|------|
| 按钮 hover | 轻微上浮 + 阴影加深 | 150ms |
| 卡片 hover | 轻微 3D 倾斜 + 阴影 | 200ms |
| 上传成功 | 照片「落入」相框动画 | 400ms |
| 风格化生成中 | 星星旋转 / 魔法粒子 | 持续 |
| 插画生成完成 | 单页 ✅ 弹跳出现 | 300ms |
| 全部完成 | Confetti 庆祝 + 绘本封面弹出 | 1s |
| 模板选中 | 发光边框 + 轻微弹跳 | 250ms |
| 错误提示 | 轻微抖动 | 300ms |
| Toast 通知 | 从右上角滑入 + 自动消失 | 3s |

### 6.3 加载状态

| 场景 | 设计 |
|------|------|
